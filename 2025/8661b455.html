1. 亲电指数（Electrophilicity Index, ω）简介

亲电指数（ω）是描述分子或自由基亲电性强弱的量化指标，反映其吸引电子的能力。计算公式为：

其中：

  • 化学势（μ）：体系得电子能力的度量，μ 越小亲电性越强。
  • 硬度（η）：体系抵抗电荷变化的能力，η 越小化学反应性越高。

亲电性排序：ω 值越大，亲电性越强（如 SO₄•⁻ > OH• > CO₃•⁻）。

2. 计算公式与物理量含义

计算步骤（推荐有限差分法）：

  1. 电离能（I）：移去一个电子所需能量
    [ I = E(N-1) - E(N) ]
  2. 电子亲和能（A）：得到一个电子释放的能量
    [ A = E(N) - E(N+1) ]
  3. 化学势（μ）：
    [ \mu = -\frac{I + A}{2} \quad \text{(eV)} ]
  4. 硬度（η）：
    [ \eta = I - A \quad \text{(eV)} ]
  5. 亲电指数（ω）：
    [ \omega = \frac{\mu^2}{2\eta} \quad \text{(eV)} ]

5.注意事项

  1. 溶剂一致性：所有计算必须使用相同的溶剂模型（如 scrf=(smd,solvent=water)）。
  2. 自旋污染检查：对开壳层体系（如自由基），检查输出中 <S²> 是否接近理论值（双电子自由基为 0.75）。
  3. 二价阴离子稳定性：如 SO₄²⁻ 需在溶剂模型中计算，气相可能不收敛。
  4. 坐标格式：单点计算建议用 geom=check 直接读取优化后的结构。
